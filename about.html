About Me

Hello! My name is Ali & Gheed and we are students in I2201. This project showcase 5 unique monuments around the world, combining history, culture, and architecture.

Project purpose

The purpose of this project is to practice building a fully responsive multi-page website using HTML, CSS and JavaScript, including interactivity and client-side validation.

Tools used

  • HTML5 for semantic structures
  • CSS3 for layout, styling, and responsiveness
  • JavaScript for gallery interactivity, navigation highlights, and form validation
  • Images from free public domain source
